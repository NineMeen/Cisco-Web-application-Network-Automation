Welcome to Cisco Network Automation

How to Use This Project

  1. Add your Cisco devices using the "Device" section under Router, SwitchL2, or SwitchL3.
  2. Configure Access Control Lists (ACLs) for your routers and layer 3 switches.
  3. Set up DHCP pools for automatic IP address assignment.
  4. Configure VLANs and port settings for layer 2 and layer 3 switches.
  5. Use the Import/Export Backup feature to manage device configurations.

Setting Up SSH on Cisco Devices

Steps to Enable SSH:
  1. Configure a hostname: 
    hostname your-device-name
  2. Set up a domain name: 
    ip domain-name your-domain.com
  3. Generate RSA key pair: 
    crypto key generate rsa
modulus 2048
  4. Create a user account: 
    username admin privilege 15 secret your-password
  5. Configure SSH version 2: 
    ip ssh version 2
  6. Enable SSH on VTY lines: 
    line vty 0 4
transport input ssh
login local

Getting Started

To begin automating your Cisco network:

  1. Ensure all your devices have SSH enabled using the guide above.
  2. Add your devices to the system using the appropriate device type (Router, SwitchL2, SwitchL3).
  3. Start configuring your network using the various tools provided in the sidebar.

For more detailed information, please refer to the documentation or contact support.
